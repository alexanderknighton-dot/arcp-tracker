Upload your ePortfolio export

Export your portfolio from the RCOG ePortfolio as a PDF, then drop it here. Your file is sent to the Claude API for parsing — nothing is stored on any server.

How to export from ePortfolio

  1. Log in to eportfolio.rcog.org.uk
  2. Go to My Portfolio → Export
  3. Select Full export as PDF and download
  4. Drop the PDF on the box above

Your key is stored only in this browser session and never sent anywhere except directly to the Anthropic API.
Get a key at console.anthropic.com.

Your portfolio PDF is sent only to the Anthropic API (claude.ai infrastructure) for text extraction. No data is stored, logged, or shared.
